# 参考面板页面结构

来源：参考演示站已登录管理员账号后的真实页面。

说明：

- 路由模式为 hash：`#/xxx`。
- UI 使用 Ant Design，左侧为菜单，右侧为页面主体。
- 以下为只读观察结果，未执行保存、删除、清空、充值、下单等会改变数据的动作。

## 顶部与布局

- 顶栏：
  - 左侧菜单折叠按钮。
  - 站点名：`demo`。
  - 当前用户：`admin`。
  - 用户菜单按钮。
  - 主题切换按钮。
- 侧边栏：
  - 用户侧菜单。
  - 管理侧菜单，管理员登录后展开。

## 菜单与路由

| 菜单 | 路由 | 说明 |
| --- | --- | --- |
| 主页 | `#/` | 首页公告、版本、站点/后端信息入口 |
| 个人中心 | `#/userinfo` | 用户资料、套餐、流量、钱包、Telegram、账户设置、邀请 |
| 转发规则 | `#/forward_rules` | 用户自己的转发规则 |
| 单端隧道 | `#/device_group` | 用户自己的设备组/单端隧道 |
| 商城 | `#/shop` | 钱包充值、购买套餐、兑换套餐 |
| 我的订单 | `#/orders` | 用户订单列表 |
| LookingGlass | `#/looking_glass` | 诊断入口，按设备组展示 |
| 节点状态（旧） | `#/tz` | 旧节点状态页，演示站当前无主体数据 |
| 节点状态（新） | `#/tz2` | 新节点状态页，演示站当前无主体数据 |
| 管理 / 仪表盘 | `#/admin/main` | 管理端数据总览 |
| 管理 / 站点设置 | `#/admin/settings` | 基本配置、公告、支付、邀请 |
| 管理 / 推送通知 | `#/admin/push_settings` | Telegram Bot、设备离线通知 |
| 管理 / 用户管理 | `#/admin/users` | 用户列表与用户规则搜索 |
| 管理 / 用户组管理 | `#/admin/user_group` | 用户组列表 |
| 管理 / 订单管理 | `#/admin/orders` | 全站订单管理 |
| 管理 / 套餐管理 | `#/admin/plans` | 套餐列表 |
| 管理 / 兑换码管理 | `#/admin/redeem` | 兑换码列表 |
| 管理 / 设备组管理 | `#/admin/device_group` | 管理员设备组列表 |
| 管理 / 邀请记录 | `#/admin/afflog` | 返佣/邀请记录 |
